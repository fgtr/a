`跑` 
+ `头` 保持脸和身体在一条直线上，不要刻意往前伸或向后倾。
+ `肩` 不要让肩处于紧缩状态，尽量使之放松。处于没有压力，且保持水平的状态。
+ `手臂` 微微握拳，手臂尽可能幅度大地前后垂直摆动，摆动速度不要太快。
+ `身体` 背挺直，自然的挺直就可以了。
+ `脚` 脚底的中间部位先着地，然后是前面部位，再到脚跟。尽可能地扩大脚与地面的接触面积。膝盖和脚尖保持在同一直线上。
+ `顶点` 要了解身体的一个顶点。经过很长一段距离的跑步之后，会出现呼吸困难、四肢无力的状态，这就是你运动的一个顶点。但遇到这个顶点时不要停止运动，你可以稍微放慢速度，依赖自己的毅力继续跑步，过了一小段时间，你就能挺过这个点，坚持跑下去。
+ 把长跑定义为一项强度适中的训练运动，也就是说每次跑到7、8分钟左右时，才开始出汗。
+ *用腹部呼吸，如果感到呼吸沉重，那就跑慢一点或者走一会儿。*
+ *制定一个跑步计划，并不要轻易改变，否则会让你感到懊恼与沮丧。*
+ *不要漫无目地的跑步，认真记录你的跑步日记，适当地进行调整，每周增加跑量不要超过10%。*
+ **跑步1小时内，是补充食物和水的最佳时间，不要错过**，但要适量，否则是很容易长胖的。
